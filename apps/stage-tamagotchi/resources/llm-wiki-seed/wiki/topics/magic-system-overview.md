# Magic System Overview

## 核心概览

《葬送的芙莉莲》的魔法系统并不只是“招式列表”，而是由历史传承、个人偏好、考试制度与组织体系共同构成。

## 传承脉络

- [[flamme]] 是理解古代传承与人类魔法发展的关键人物。
- [[frieren]] 同时连接“古老魔法知识”与“现代理解方式”。
- [[serie]] 则代表更高层级、更古老且更苛刻的魔法标准。

## 制度与组织

- [[continental-magic-association]] 提供了魔法师制度化、组织化的框架。
- [[first-class-mage-exam]] 则是理解现代魔法评价标准与强者分层的主要入口。

## 魔法的使用方式

- [[spells]] 是具体魔法条目的入口。
- 作品中的魔法不仅用于战斗，也常体现角色习惯、价值判断与生活方式。

## 关键观察

- 同样会用魔法，不同角色的战斗思路和价值取向并不相同。
- 魔法系统在叙事上既是战斗工具，也是角色性格与时代差异的表达方式。
