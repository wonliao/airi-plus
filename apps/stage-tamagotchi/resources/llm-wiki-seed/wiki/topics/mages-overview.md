# Mages Overview

## 核心概览

《葬送的芙莉莲》的魔法使角色并不只服务于战斗，而是构成了不同世代、不同制度与不同战斗观的横截面。

## 观察重点

- [[flamme]] 与 [[serie]] 代表更古老、更高层级的魔法传承。
- [[frieren]] 与 [[fern]] 展示主线视角下的师徒与继承关系。
- 考试篇角色则让“现代魔法使”呈现出更鲜明的个体差异。
