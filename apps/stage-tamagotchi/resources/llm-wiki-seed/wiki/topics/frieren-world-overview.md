# Frieren World Overview

## 核心概览

《葬送的芙莉莲》的世界观可分成几条稳定主线：角色群像、队伍关系、魔法体系、魔族势力、地理旅程，以及漫画/动画/音乐等媒介信息。

## 角色与队伍

- 角色核心以 [[frieren]]、[[fern]]、[[stark]] 为当前旅程主轴。
- 历史记忆的核心则围绕 [[hero-party]]，其中包括 [[himmel]]、[[heiter]]、[[eisen]] 与 [[frieren]]。
- 当代行动主线可看作 [[frieren-s-party]]，它承接了过去旅程的情感与目标。

## 魔法与组织

- 魔法体系的关键入口包括 [[spells]]、[[first-class-mage-exam]] 与 [[continental-magic-association]]。
- 在角色层面，[[flamme]]、[[serie]]、[[sense]]、[[genau]]、[[lernen]] 等人构成了魔法知识、传承与考核制度的不同面向。

## 魔族与冲突

- 世界层面的主要敌对结构由 [[demon-king]]、[[seven-sages-of-destruction]]、[[greater-demons]] 与 [[list-of-demons]] 共同组成。
- 个体层面可从 [[aura]]、[[macht]]、[[bose]]、[[grausam]]、[[solitar]]、[[tot]] 等条目切入。

## 地理与旅程

- 旅程范围可通过 [[locations]]、[[northern-lands]]、[[central-lands]]、[[southern-lands]] 理解。
- 关键地点与叙事目标则可补充参考 [[aureole]]、[[ende]]、[[royal-capital-strahl]]。

## 媒体与外部资料

- 漫画与动画入口见 [[manga]]、[[anime]]、[[season-1]]、[[season-2]]。
- 音乐与歌曲整理见 [[music]]、[[the-brave]]、[[sunny]]、[[anytime-anywhere]]、[[bliss]]、[[original-soundtrack]]。

## 建议阅读顺序

1. 先读 [[frieren]]、[[hero-party]]、[[frieren-s-party]]。
2. 再读 [[first-class-mage-exam]]、[[continental-magic-association]]、[[spells]]。
3. 最后扩展到 [[demon-king]]、[[seven-sages-of-destruction]]、[[locations]] 与媒体相关条目。
