# Frieren Relationship Map

## 核心概览

《葬送的芙莉莲》的角色关系可从三条主轴理解：过去的勇者一行人、当代旅程队伍，以及魔法协会与考试篇的人物网络。

## 过去的关系核心

- [[frieren]] 与 [[himmel]]：情感理解与回望的核心关系。
- [[frieren]] 与 [[heiter]]、[[eisen]]：勇者一行人的长期同行关系。
- [[flamme]] 与 [[frieren]]：师徒与魔法传承关系。

## 当代的关系核心

- [[frieren]] 与 [[fern]]：近似师徒、照顾者与继承者的关系。
- [[fern]] 与 [[stark]]：当前旅程中最稳定的人际互动线之一。
- [[sein]] 与 [[frieren-s-party]]：补足队伍中的治愈与观察视角。

## 魔法体系的人物网络

- [[serie]]、[[lernen]]、[[sense]]、[[genau]]：构成现代魔法制度与评价体系的关键节点。
- [[denken]]、[[ubel]]、[[land]]、[[wirbel]]：体现考试篇中的竞争、协作与个体差异。

## 魔族对立关系

- [[aura]]、[[macht]]、[[lugner]] 等角色与主角方的关系更多体现为冲突与价值观对照。
- [[demon-factions-overview]] 可作为魔族人物关系的延伸入口。
